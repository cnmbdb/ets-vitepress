# 开发计划 (Roadmap)

我们将分三个阶段逐步完善应用商店生态。

## Phase 1：MVP (最小可行性产品)
**周期：4-6周**

*   ✅ **开发者入驻**：基础注册流程，实名认证。
*   ✅ **应用管理**：简单的上传、编辑和上架流程。
*   ✅ **商店前台**：基础的浏览、搜索和详情页展示。
*   ✅ **人工审核**：后台审核队列，手动处理上架申请。
*   ✅ **基础购买**：支持余额购买和一键安装。

## Phase 2：生态完善
**周期：8-10周**

*   🚧 **自动化审核**：引入代码静态扫描工具，提升审核效率。
*   🚧 **评价系统**：用户评分与评论功能，建立信用体系。
*   🚧 **数据分析**：开发者数据看板，展示下载量与收入。
*   🚧 **推荐算法**：基于用户行为的热门应用推荐。
*   🚧 **结算系统**：自动化的开发者收入结算与提现流程。

## Phase 3：高级功能
**周期：6-8周**

*   📅 **应用内购买 (IAP)**：支持应用内解锁高级功能。
*   📅 **订阅制**：支持按月/按年付费的 SaaS 模式应用。
*   📅 **组合包 (Bundles)**：多个应用打包优惠购买。
*   📅 **API 市场**：不仅是应用，还支持 API 接口的交易。
*   📅 **协作工具**：支持团队开发，多人管理同一个应用账号。
