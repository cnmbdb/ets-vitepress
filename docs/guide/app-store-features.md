# 功能模块详解

## 1. 应用商店前端（用户侧）

商户在独角数卡后台可见的界面，提供类似 App Store / Google Play 的交互体验。

### 核心功能
*   **应用发现**
    *   **首页推荐**：展示热门应用、最新上架、编辑精选。
    *   **分类浏览**：按功能分类（如支付、模板、工具、营销）。
    *   **搜索与筛选**：支持关键词搜索，按价格、评分、热度排序。
*   **应用详情**
    *   **基础信息**：应用图标、名称、开发者、版本号、更新时间。
    *   **详细介绍**：富文本描述、功能亮点。
    *   **视觉展示**：应用截图、演示视频。
    *   **用户评价**：评分统计、用户评论列表。
    *   **版本历史**：更新日志（Changelog）。
*   **购买与安装**
    *   **购买流程**：支持余额支付、在线支付，支持应用内购买。
    *   **一键安装**：购买后自动下载、解压、安装依赖、执行数据库迁移。
*   **已购管理**
    *   **应用列表**：查看已安装和已购买的应用。
    *   **状态管理**：启用/停用、卸载、更新。
    *   **更新通知**：当有新版本时，自动提示商户进行更新。

### 设计要求
*   **响应式设计**：完美适配 PC 端和移动端管理后台。
*   **隐私透明**：在安装前清晰展示应用所需的系统权限（如读取订单、修改商品等）。

## 2. 开发者中心

供第三方开发者注册、管理应用和查看收益的独立后台。

### 核心功能
*   **注册与认证**
    *   支持个人开发者和企业开发者入驻。
    *   实名认证机制，确保开发者身份真实可靠。
*   **应用管理**
    *   **创建应用**：填写应用元数据（名称、标识符、分类）。
    *   **版本发布**：上传安装包（ZIP）、填写版本号、更新说明、兼容性要求。
    *   **定价设置**：设置免费、一次性付费或订阅制价格。
    *   **状态控制**：申请上架、下架、暂停销售。
*   **数据仪表盘**
    *   **销售数据**：每日/每月下载量、收入趋势图。
    *   **用户分析**：活跃安装量、留存率（规划中）。
    *   **评价管理**：查看并回复用户评价。
*   **审核系统**
    *   **自动扫描**：上传代码后自动进行安全扫描（SQL注入、XSS、敏感函数）。
    *   **人工审核**：平台管理员对应用功能和安全性进行最终复核。

## 3. 运营管理后台（平台侧）

平台管理员使用的管理系统。

*   **应用审核**：查看待审核队列，下载代码包，通过或驳回申请。
*   **开发者管理**：审核开发者资质，封禁违规开发者。
*   **推荐管理**：配置商店首页的推荐位和广告位。
*   **财务结算**：查看交易流水，处理开发者提现请求，配置平台抽成比例。
