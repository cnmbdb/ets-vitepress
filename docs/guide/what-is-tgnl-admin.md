# 什么是 TGNL Admin？

TGNL Admin 是一个功能强大的 **Telegram Bot 管理后台系统**，基于 Nuxt 3 构建，提供了完整的机器人管理、用户管理、订单管理等功能。

## 使用场景

### • 文档

TGNL Admin 提供了完整的文档系统，帮助开发者快速了解和使用系统。你现在正在阅读的这个文档站点就是基于 VitePress 构建的。

### • 机器人运营

TGNL Admin 专为 Telegram Bot 运营而设计，提供了：
- 多机器人管理
- 用户数据统计
- 订单处理流程
- 自动化配置

### • 数据分析

系统内置了强大的数据分析功能：
- 实时数据监控
- 图表可视化
- 报表导出
- 趋势分析

## 开发体验

### 快速启动

使用 Docker 一键部署，几分钟内即可启动完整的系统：

```bash
docker compose up -d
```

### 模块化设计

系统采用模块化设计，功能清晰，易于扩展和维护。

### 完善的文档

提供详细的文档和 API 参考，帮助开发者快速上手。

## 性能

### 快速响应

- 基于 Nuxt 3 的 SSR 渲染
- 优化的数据库查询
- 高效的缓存机制

### 可扩展性

- 支持多机器人管理
- 可处理大量并发请求
- 易于水平扩展

## 技术栈

- **前端框架**: Nuxt 3
- **UI 组件**: Nuxt UI (基于 Tailwind CSS)
- **状态管理**: Pinia
- **数据库**: MySQL
- **容器化**: Docker & Docker Compose
