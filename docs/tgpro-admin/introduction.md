# 项目介绍

**TGPro Admin** 是一套功能强大的 Telegram 机器人管理系统，旨在为用户提供一站式的机器人运营解决方案。

## <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="title-icon"><polygon points="12 2 2 7 12 12 22 7 12 2"></polygon><polyline points="2 17 12 22 22 17"></polyline><polyline points="2 12 12 17 22 12"></polyline></svg> 核心功能

- **多币种支付支持**：原生支持 USDT (TRC20) 和 TON 支付，满足不同用户的支付习惯。
- **自动发货系统**：用户支付成功后，系统自动检测并即时发货，无需人工干预。
- **强大的后台管理**：提供可视化的 Web 管理后台，轻松配置机器人参数、查看订单状态、管理用户数据。
- **数据统计分析**：直观的仪表盘展示，实时掌握今日订单、总流水、新增用户等关键运营指标。
- **消息广播功能**：支持向所有用户或特定用户组发送广播消息，高效触达用户。

## <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="title-icon"><path d="M12 22s8-4 8-10V5l-8-3-8 3v7c0 6 8 10 8 10z"></path></svg> 安全与稳定

- **私钥本地存储**：支付私钥仅存储在您部署的本地服务器中，确保资金绝对安全。
- **Docker 容器化**：基于 Docker 架构，服务隔离，部署简便，迁移无忧。
- **高性能架构**：后端采用 Go 和 Python 混合开发，前端采用 Nuxt.js，兼顾开发效率与运行性能。

## <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="title-icon"><path d="M4 19.5A2.5 2.5 0 0 1 6.5 17H20"></path><path d="M6.5 2H20v20H6.5A2.5 2.5 0 0 1 4 19.5v-15A2.5 2.5 0 0 1 6.5 2z"></path></svg> 适用场景

- **数字商品售卖**：自动售卖激活码、账号、会员等虚拟商品。
- **付费群组管理**：支付后自动拉入 VIP 群组或频道。
- **内容订阅服务**：提供付费内容订阅和自动推送服务。
