项目介绍 ​

Telegram 五合一机器人 是一款全开源、无加密的综合型运营机器人，集成了 TRX 兑换、能量租赁、会员充值、钱包监控 和 发卡系统 五大核心功能。配合独立的 PHP 后台，您可以轻松管理所有业务，是 Telegram 生态变现的终极利器。

2024.10.1 最新版

我们已与原创作者深度合作推出平台专属版，修复了已知问题，并支持独立部署 API 端。

五大核心模块 ​

1. TRX 兑换系统 ​

  • 自动兑换：支持 USDT 与 TRX 的双向自动兑换。
  • 汇率管理：后台可实时调整汇率差价，实现自动化盈利。

2. 能量租赁平台 ​

  • 灵活模式：支持自有质押能量，也支持对接第三方 API 平台。
  • 轮询机制：支持多平台 API 轮询，保障能量供应的稳定性。

3. Telegram 会员充值 ​

  • 官方接口：采用官方正规接口，安全稳定。
  • 自定义套餐：针对不同的会员平台或时长，设置差异化的价格套餐。

4. 钱包地址监控 ​

  • 实时通知：监听指定钱包地址的资金变动，并发送实时通知。
  • 付费服务：可将监控功能作为付费服务出售给用户。

5. 综合发卡系统 ​

  • 虚拟商品：支持卡密、账号等各类虚拟商品的自动售卖与发货。
  • 库存管理：完善的库存管理与补货提醒。

技术特性 ​

  • 全开源：无任何加密文件，支持二次开发。
  • PHP 后台：成熟稳定的 PHP 架构，易于维护。
  • 高度自定义：命令、键盘、关键词回复、定时广告均可自由配置。
  • 多机器人：一套系统可管理多个机器人实例。
