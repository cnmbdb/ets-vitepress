使用手册 ​

后台管理 ​

部署完成后，您可以访问 http://您的域名/admin 进入管理后台。

  • 默认账号：admin
  • 默认密码：123456 (请及时修改)

功能配置 ​

TRX 兑换设置 ​

  1. 进入后台 -> 兑换管理。
  2. 设置汇率浮动比例（例如：赚取 2% 差价）。
  3. 配置资金归集钱包地址。

能量租赁配置 ​

  1. 进入后台 -> 能量管理。
  2. 添加您的 API 供应商（如：trongas, feiyu 等）。
  3. 设置轮询策略，确保高可用。

会员充值对接 ​

  1. 进入后台 -> 会员管理。
  2. 配置官方会员充值接口密钥。
  3. 设置前端售卖价格。

消息通知 ​

您可以配置机器人将重要通知（如订单成交、余额不足）发送到指定的 Telegram 群组或频道，方便团队协作管理。
