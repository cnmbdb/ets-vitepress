Руководство по использованию ​

Это руководство расскажет вам, как использовать панель администратора в качестве администратора и как пользователи взаимодействуют с Telegram-ботом.

Команды бота (Сторона пользователя) ​

Общие команды для пользователей, взаимодействующих с Telegram-ботом:

  • /start - Запустить бота, просмотреть главное меню
  • Купить энергию - Нажмите кнопку меню, выберите продолжительность и сумму
  • Проверить заказ - Запросить статус последних заказов на аренду
  • Центр пользователя - Просмотреть баланс, реферальную ссылку и т.д.
  • Связаться с поддержкой - Получить контактную информацию службы поддержки

Панель администратора (Сторона администратора) ​

После доступа к адресу панели администратора и входа в систему вы можете увидеть следующие модули:

1. Панель управления ​

Просмотр общих данных системы, включая количество заказов за сегодня, общий оборот, новых пользователей и т.д.

2. Управление заказами ​

  • Заказы на энергию: Просмотр записей об аренде энергии всех пользователей, поддержка фильтрации по статусу (Успех, Ошибка, В процессе).
  • Компенсировать заказ: Для неудачных заказов вручную попытайтесь компенсировать или вернуть средства.

3. Управление пользователями ​

  • Просмотр списка всех пользователей, которые использовали бота.
  • Изменение баланса пользователя (Пополнение/Списание).
  • Бан пользователей-нарушителей.

4. Управление ботом ​

  • Перезапустить бота: Нажмите эту кнопку, чтобы применить изменения после изменения конфигурации.
  • Push-сообщение: Рассылка сообщений всем пользователям или определенным пользователям.

5. Интеграция пула энергии ​

  • Создать учетную запись пула энергии: Создать API-ключ для внешних вызовов.
  • Интегрировать сторонний пул: Настроить информацию API вышестоящего пула энергии.

Часто задаваемые вопросы (FAQ) ​

В: Что делать, если бот не отвечает? ​

О:

  1. Проверьте, нормально ли работают контейнеры Docker (docker compose ps).
  2. Проверьте логи контейнера bot (docker compose logs -f bot) на наличие ошибок.
  3. Убедитесь, что токен правильный и сеть сервера может подключиться к API Telegram.

В: Энергия не получена? ​

О:

  1. Проверьте, достаточно ли на адресе выплаты TRX для комиссии за транзакцию (если необходимо).
  2. Проверьте, достаточно ли в пуле энергии свободной энергии.
  3. Просмотрите сообщения об ошибках в деталях заказа на панели администратора.

Совет по обслуживанию

Регулярно делайте резервную копию базы данных (nl-admin.sql или каталог /var/lib/mysql), чтобы предотвратить потерю данных.
